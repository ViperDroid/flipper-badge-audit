# Security Policy

Badge Audit is a **defensive, read-only** tool. Use it only on credentials and
systems you own or are explicitly authorized to assess.

## Reporting a vulnerability

If you find a security issue in this app, please **do not** open a public issue.
Instead, report it privately via GitHub's **"Report a vulnerability"** button
under the **Security** tab, or contact the maintainer directly.

Please include:
- a description of the issue and its impact,
- steps to reproduce, and
- the app version (see `application.fam`).

You can expect an initial response within a few days.
