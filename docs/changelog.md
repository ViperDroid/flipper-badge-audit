## 1.0

- Initial release.
- Identifies an NFC badge's protocol (MIFARE Classic / Ultralight / DESFire / Plus, ISO14443-3A/4A/B, ISO15693, FeliCa, ST25TB).
- Reads and displays UID, SAK and ATQA.
- Tests every MIFARE Classic sector against common factory/default keys and reports how many are unprotected.
- Clone-resistance score (0-100) with WEAK / MEDIUM / STRONG / CRITICAL verdict.
- Exports a posture scorecard to the SD card.
- Rescan without leaving the app.
